<h3 align="center">Resumo do Projeto</h3>
<h6>API simples em Node.js com Express que expõe um catálogo de animes para consulta e cadastro. O serviço mantém uma lista em memória com atributos básicos (id, nome, gênero, protagonista) e oferece endpoints para verificar o status da API, listar todos os animes, consultar por id e adicionar novos registros. Foi pensada como uma aplicação didática para aprender conceitos de rotas REST, manipulação de JSON e testes com clientes HTTP como Postman. 

</h6>
